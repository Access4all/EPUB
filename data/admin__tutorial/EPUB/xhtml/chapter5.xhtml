5. Interactive activities

It's now time to look at interactive activities. We will see two of them here, but don't hesitate to try out other ones.

5.1. Multiple choice quiz

This is probably the most known and the most frequent interactive activity. To create a new multiple choice quiz, choose its type in the New file tab.

Go to the Quiz time! page to have an example which to experiment with.

5.2. Fill in the gaps

This is another form of activity which is well known: a text with gaps to fill in.

Select this type in the new file tab, or see an example in the Gap time! page.
