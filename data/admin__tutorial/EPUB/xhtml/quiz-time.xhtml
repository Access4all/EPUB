Quiz time!

This is an example multiple choice quiz. Use the tab key to navigate through questions, answers, and checkboxes. New answers and questions are automatically added if you press tab when you are at the end.

You could try to do the following things :

  • The correct answer of the first question is absent and a wrong one is checked. Add it as its last proposition and mark it as being the real answer.
  • Add a new question What is the color of the sky, three wrong answers as well as the correct one; the right answer should be the 3rd one.
  • People living in Basel are furious with the second question; they want their city to be proposed even if it's the wrong answer. Add it after Geneva but before Lausanne.
  • We would like to add a new question Ammong the following animals, which ones are mammals, with the following answers: elephant, butterfly, cat, frog, bee, girafe. Do you see where is the problem? You can solve it by going to Page options and change the type of quiz.
  • You can also add icons and inline formatting such as bold within the text of questions and answers if you like to. However, in that case it's better to use keyboard shortcuts rather than use the toolbar or drag and drop. To prove it, add a question On which of these sentances the subject is unproperly emphasized, write 4 different sentances as answers and mark each time the subject of the sentance in italic, as the question suggests. There are much better ways to ask such a question, but that's just an exercise.

Have fun !
